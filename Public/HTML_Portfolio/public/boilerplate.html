HTML Boiler Plate

This website displays the most basic boiler plate code for an html file:



            <!DOCTYPE html>
            <html lang="en">
            
                <head>
                    <meta charset="UTF-8" />
                    <title>My website</title>
            
                </head>
            

                <body>
                    <!-- here the actual code comes-->

                </body>
            
            </html>
